# CareOS Design Mockups

Interactive design mockups for the CareOS platform — an AI-powered eMAR (electronic Medication Administration Record) system for assisted living facilities.

## 🖥️ Live Demo

Visit: `https://YOUR_USERNAME.github.io/careos-mockups/`

## 📱 13 Interactive Screens

### Core Flows
- **Med Pass** — 3-click medication administration workflow (queue → confirm → complete)
- **AI Order Processing** — Photo capture → AI extraction → review with confidence scores
- **Medication Receipt & Verification** — Pending deliveries, 3-way reconciliation, inventory tracking

### Management
- **Resident Profile** — 5-tab view (overview, meds, care plan, vitals, notes)
- **Add New Resident** — 4-step wizard with progress tracking
- **Staff Scheduling** — Weekly shift grid with coverage alerts

### Dashboard
- **Owner Dashboard** — Web layout with multi-home monitoring, alerts, credentials
- **Inspection Readiness** — DSHS compliance scoring by category

### App
- **Notifications** — Filterable alert feed with unread indicators
- **Settings** — Account, preferences, compliance, support sections
- **User Profile** — Stats, contact info, credentials

### Marketing
- **Onboarding** — 3-slide feature carousel + signup flow
- **App Store Preview** — Listing mockup with ratings and reviews

## 📲 Install on Phone

1. Open the live URL in Safari (iOS) or Chrome (Android)
2. Tap **Share → Add to Home Screen**
3. The app will run in standalone mode

## Tech Stack

- React 18 (loaded via CDN)
- Babel for in-browser JSX compilation
- PWA manifest for home screen install
- Service worker for offline caching
- DM Sans typography via Google Fonts

## Design System

- **Primary**: `#1A6B8A` (Deep teal — trust, calm)
- **Accent**: `#E8734A` (Warm coral — attention, action)
- **Success**: `#2D9E6E` / **Warning**: `#D4890B` / **Danger**: `#C43D3D`
- **Philosophy**: Caregiver-first, mobile-first, 3-click workflows
